# Musician Portfolio (React + TS + Vite)

## Run locally
1. Install dependencies:
   - `npm install`

2. Start dev server:
   - `npm run dev`

3. Build:
   - `npm run build`
   - `npm run preview`

## Customize
- Update Spotify link + artist ID in `src/App.tsx`
- Replace placeholder email + socials in `src/App.tsx`
- Swap images in `src/assets/` and update the imports in `src/App.tsx`

> Note: `DSC00793.ARW` is included in assets but not used (RAW files aren’t web-friendly).
